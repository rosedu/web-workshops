ROSEdu WebDev – workshop-uri despre web

Web e cool!

Aplicațiile moderne de astăzi sunt scrise pentru web. E o platformă universală, gata instalată pe orice calculator, accesibilă pentru toată lumea, pornind de la un link.

Browser-ul web este un suport grozav pentru construirea interfeței cu utilizatorul. Aspectul vizual este controlat până în cele mai fine detalii folosind stylesheet-uri. Interacțiunea e definită prin cod JavaScript ușor de scris și de înțeles. Workflow-ul este cum nu se poate mai simplu: modifici un fișier, reîncarci pagina, și vezi rezultatul, nu aștepți după vreun compilator. Singurul IDE de care ai nevoie este un editor de text.

Versiuni noi ale aplicației ajung în mâna utilizatorului imediat: actualizezi codul de pe server și gata! Bug-urile sunt reparate pentru totdeauna și toată lumea se bucură de ultimele feature-uri.

Programatorii web sunt în linia întâi a inovației. Au nevoie de cele mai avansate tehnologii ca să construiască lucruri precum GitHub, Wikipedia, Gmail și AWS.

Tehnologiile web sunt într-o evoluție continuă. Browserele web moderne suportă grafică vectorială (SVG), grafică bitmap 2D (canvas) și 3D (WebGL), comunicare în timp real cu serverul (WebSocket), stocarea de date (web storage), funcționare offline, geolocație, markup mai bogat și opțiuni mai puternice de formatare a conținutului.
